##Requirements
- Windows
- Preinstalled Java 17 or Bundled version with Launch4j

## Technical information

- Java: 17
- JUnit: 5.8.2
- Gradle: 7.2
- Log4j: 2.17.1
- Lombok: 1.18.22
- YOML (Java OpenGl Math Library): 1.10.3
- LWJGL (Lightweight Java Game Library): 3.3.0
- Launch4j: 2.5.1
